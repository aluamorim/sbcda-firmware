void HK_Set_Config ( uint16_t  word)

Sets the current HK config word.

HK_Set_Config

The configuration word masks defines which HK measure is enabled or not. It can be updated by the on-board computer using the HK_CONFIG command with the firmware in Configuration Mode. The configuration word is a 2-byte integer whose bits are mapped as follows: 

0 - RTC time tag
1 - Current sensors
2 - ADC RMS
3 - PLL Synch flag
4 - Over - current
5-15 - Reserved for future use

wordHK configuration word.
