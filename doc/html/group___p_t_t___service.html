PTT_FREE

PTT_FRAME_SYNCH

PTT_MESSAGE_LENGTH 

Package is receiving the Message Length parameter bits.

PTT_ID_NUMBER 

Package is receiving the PTT ID Number parameter bits.

PTT_DATA 

Package is receiving the PCD Sensor data bits

PTT_ERROR 

Package failed the Fram Synch phase.

PTT_READY 

Package is is ready to be transmited to the on-board computer.
